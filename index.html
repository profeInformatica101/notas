Guía de Markdown para GitHub

Encabezados
Utiliza el símbolo # para los encabezados. 
# Encabezado 1
## Encabezado 2
### Encabezado 3
Énfasis

Para poner negritas, usa ** o __.

**texto en negrita**
__texto en negrita__

Para poner *cursiva*, usa * o _.

*texto en cursiva*
_texto en cursiva_
Enlaces

Puedes crear enlaces de la siguiente forma:

[Texto del enlace](https://www.enlace.com)
Listas

Listas no ordenadas:

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

Listas ordenadas:

1. Primer item
2. Segundo item
3. Tercer item
Imágenes

Para añadir imágenes:

![Texto alternativo](https://www.enlacealaimagen.com/imagen.jpg)
Código

Para añadir un bloque de código:

Para código en línea, utiliza ` al principio y al final:

Esto es `código en línea`
Tablas

Puedes crear tablas de la siguiente forma:

| Encabezado 1 | Encabezado 2 |
|--------------|--------------|
| celda 1      | celda 2      |
| celda 3      | celda 4      |
Bloques de cita

Para añadir bloques de cita:

> Este es un bloque de cita
